# A To-Do CRUD Application
This is a simple todo application with CRUD functionality. It allows you to add new tasks, mark tasks as complete, edit existing tasks, and delete tasks.
# Tech Stack

- HTML
- CSS + Bootstrap 5
- EJS
- NodeJS
- Express
- MongoDB
